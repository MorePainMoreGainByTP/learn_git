为什么Git添加文件需要add，commit一共两步呢？因为commit可以一次提交很多文件，所以你可以多次add不同的文件
先将修改的文件通过 ‘add’添加到‘commit’里面（相当于暂存区），然后‘commit’再把所有的‘add’的文件提交到仓库
